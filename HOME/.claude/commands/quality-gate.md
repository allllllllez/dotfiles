---
description: Run code quality gate review using quality-gate-keeper agent
---

Please review the code quality of recent changes using the Task tool with `subagent_type='quality-gate-keeper'`.

If there are any quality issues identified:
1. Review the feedback from quality-gate-keeper
2. Fix the identified issues
3. Re-run this command if needed

This command is designed to be run manually when you want to ensure code quality before committing or completing a feature.
